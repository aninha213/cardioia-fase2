# CardioIA – Fase 2: Diagnóstico Automatizado com IA

## Descrição do Projeto
O CardioIA é um projeto acadêmico que simula um sistema de apoio ao diagnóstico médico utilizando Inteligência Artificial. O objetivo é analisar descrições de sintomas escritos por pacientes e,
a partir disso, identificar possíveis doenças e classificar o nível de risco como alto ou baixo.

O sistema utiliza técnicas de Processamento de Linguagem Natural (NLP) e Machine Learning para simular, de forma simples, o funcionamento de ferramentas usadas em triagens médicas reais.

---

## Objetivos do Projeto
- Simular um sistema de apoio ao diagnóstico clínico
- Aplicar técnicas de NLP para análise de texto
- Classificar o nível de risco de pacientes com base em sintomas
- Demonstrar o uso de Machine Learning em um contexto da saúde
- Entender de forma prática como a IA pode ser aplicada na área médica

---

## Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn
- TF-IDF (Processamento de Linguagem Natural)
- Google Colab

---

## Funcionalidades do Sistema
- Leitura de descrições de sintomas em texto
- Identificação de possíveis doenças com base em um mapa de conhecimento
- Classificação automática de risco (alto ou baixo)
- Treinamento de modelo de Machine Learning
- Avaliação de desempenho do modelo com acurácia
- Simulação de um sistema de triagem médica

---

## Etapas do Desenvolvimento

### Parte 1 – Diagnóstico por Sintomas
- Criação de frases simulando pacientes
- Construção de um mapa de doenças relacionando sintomas e diagnósticos
- Implementação de um sistema simples de identificação por palavras-chave

### Parte 2 – Classificação de Risco com IA
- Criação de um dataset com frases médicas rotuladas
- Aplicação de TF-IDF para transformar texto em dados numéricos
- Treinamento de um modelo de Machine Learning (Regressão Logística)
- Testes com novas frases
- Avaliação da acurácia do modelo

---

## Resultados
O modelo foi capaz de identificar padrões simples entre sintomas e níveis de risco, demonstrando como a inteligência artificial pode ser aplicada em problemas reais de forma simplificada.

---

## Observação
Este projeto é apenas uma simulação acadêmica e não deve ser utilizado para diagnósticos médicos reais. O objetivo é exclusivamente educacional.
